# Sharp Blade - Blender Themes

**Easy on the eyes, deep in the blue. Create freely.**

Welcome to the official repository for the **Sharp Blade** theme collection. This repository hosts the latest versions of the Sharp Blade themes for Blender, designed to provide a professional, high-contrast, and distraction-free environment for 3D artists.

Sharp Blade is designed to help you dive deep into your projects with a calming palette of blue. It minimizes eye strain during long working sessions while ensuring that active elements remain distinct and readable.

### Key Characteristics
* **Dark & Minimalist:** A refined dark background that reduces glare.
* **High Contrast Viewport:** 3D elements pop against the background for precise modeling.
* **Low Contrast Panels:** UI panels are subtle, keeping the focus on your work.
* **Accented Popups:** Menus and active tools are highlighted for quick navigation.

## ğŸ“¦ Installation

Current version can be selected directly from Blender in preferences, but you can also find it here:
[Sharp Blade v5.1.0](https://extensions.blender.org/themes/bdvd-sharpblade/).

1.  **Download** the repository or the specific `.xml` file for the version you want (e.g., inside the `Sharp Blade v5` folder).
2.  Open **Blender**.
3.  Go to **Edit** > **Preferences**.
4.  Select the **Themes** tab on the left.
5.  Click the **Install...** button at the top right.
6.  Navigate to the downloaded `.xml` file and select it.
7.  The theme will apply immediately. Remember to save your preferences!
